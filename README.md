# Overview

This repository contains the code and processed data for the analysis of mouse liver proteomics data contained in the following paper (https://pubmed.ncbi.nlm.nih.gov/39490929/). 

The zipped proteinGroups file contains proteomic quantitation while the lipogenesis_metadata.csv, contains the experimental design.

R code is provided as a R markdown file and as a html with embedded graphs.
